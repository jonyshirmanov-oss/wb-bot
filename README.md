# WB Telegram Bot — инструкция по запуску

## Что делает бот
- Принимает ссылку на товар Wildberries
- Автоматически получает: название, бренд, цену, скидку, рейтинг, фото
- Публикует красивый пост с фото в вашу Telegram-группу

---

## Шаг 1 — Получить токен бота

1. Откройте Telegram, найдите **@BotFather**
2. Напишите `/token` → выберите вашего бота
3. Скопируйте токен вида: `7123456789:AAFxxx...`

---

## Шаг 2 — Узнать ID вашей группы

1. Добавьте бота **@username_to_id_bot** в вашу группу
2. Он покажет ID группы — он начинается с минуса, например: `-1001234567890`
3. Скопируйте этот ID

---

## Шаг 3 — Загрузить на Railway (бесплатно)

### 3.1 Создайте аккаунт
- Зайдите на https://railway.app
- Войдите через GitHub (нужен аккаунт на github.com)

### 3.2 Загрузите файлы на GitHub
1. Зайдите на https://github.com → New repository → назовите `wb-bot`
2. Загрузите три файла: `bot.py`, `requirements.txt`, `Procfile`

### 3.3 Разверните на Railway
1. На Railway нажмите **New Project → Deploy from GitHub**
2. Выберите ваш репозиторий `wb-bot`
3. Нажмите **Variables** (переменные окружения) и добавьте:

| Ключ | Значение |
|---|---|
| `BOT_TOKEN` | токен из BotFather |
| `GROUP_ID` | ID вашей группы (с минусом) |

4. Railway сам установит зависимости и запустит бота!

---

## Как пользоваться ботом

1. Откройте вашего бота в Telegram
2. Напишите ему ссылку на товар WB:
   ```
   https://www.wildberries.ru/catalog/123456789/detail.aspx
   ```
3. Бот пришлёт подтверждение и опубликует пост в группу

---

## Возможные проблемы

**Бот не публикует в группу** → убедитесь что бот добавлен как **администратор** группы с правом отправки сообщений

**Ошибка токена** → проверьте что BOT_TOKEN скопирован без пробелов

**Нет фото** → некоторые товары могут не иметь фото в публичном доступе, пост опубликуется без фото
